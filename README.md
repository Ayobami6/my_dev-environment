### Download Installation Scripts

```sh
wget https://github.com/Ayobami6/simple-shell/raw/master/dev_setup.sh
```

### Plugins

```sh
wget https://github.com/Ayobami6/simple-shell/raw/master/plugins.sh
```

```sh
chmod 555 dev_setup.sh plugins.sh
```

### Run

```sh
./dev_setup.sh
./plugins.sh
```

```sh
source ~/.zshrc
```
